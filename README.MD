# API List

| Routes | EndPoint                   | Description                                |
| ------ | -------------------------- | ------------------------------------------ |
| POST   | /api/weShare/user/register | Register user                              |
| POST   | /api/weShare/user/login    | login user to get token for authentication |

---

# How to run

## Server

```bash
$ cd server
$ npm i
$ npm start
```

## Client

```bash
$ cd client
$ npm i
$ npm start
```
